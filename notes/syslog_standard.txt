# Syslog
[UDP/514]
Protocol and supporting software that facilitates log collection. It has become a de facto standard for 
logging events from distributed systems.

For example, Syslog messages can be supported by Cisco routers and switch, as well as UNIX and Linux servers 
and workstations.

Syslog provides an open format for event data. A syslog message comprises a PRI code, a header containing:

	- Timestamp
	- Hostname
	- Message

The format of the content is application dependent. It can be:

	- Comma or space delimited
	- Name-Value pairs such as JSON
